## 2、激活步骤



打开应用程序然后用编辑器打开以下路径：

```bash
/Applications/Typora.app/Contents/Resources/TypeMark/page-dist/static/js/LicenseIndex.180dd4c7.5b0f7af9.chunk.js
```

然后找到以下判断

```bash
hasActivated="true"==e.hasAgtivated
```

修改为以下内容

```bash
hasActivated="true"=="true"
```

然后重启就可以看到激活了
